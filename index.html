ezMobile POS


ezMobile POS သည် Android ဖုန်းအတွက်ထုတ်လုပ်ထားသော POS system application တစ်ခုဖြစ်ပါသည်။ ဖုန်းဖြင့်ဆိုင်ရဲ့ အသုံးစားရိတ်၊အရောင်းအ၀ယ်စာရင်းတွေကိုလွယ်ကူစွာ စီမံတွက်ချက်ပေးမှာဖြစ်ပြီးတော့ ဖော်ပြပါ featureများစွာပါ၀င်ပါတယ်။

တစ်သက်သာအသုံးပြုနိုင်မယ့် အထူးစျေးနှုန်း

တစ်သောင်းကျပ်

ဖြင့်သာ ၀ယ်ယူလိုက်ပါ


Features:









Full Features


  • ဖောက်သည်များ(Customer)နှင့်ပေးသွင်းသူများ(Supplier)၏အချက်အလက်များကိုထည့်သွင်း၊ပြင်ဆင်နိုင်ခြင်း။
  • ကုန်ပစ္စည်း၊ရောင်းကုန်တို့၏ အချက်အလက်ကိုပုံနှင့်တကွထည့်သွင်းပြင်ဆင်နိုင်ခြင်း။
  • PDF ပြေစာထုတ်နိုင်ခြင်း။
  • ပြေစာများအား ပရင့်ထုတ်နိုင်ခြင်း။
  • PDFပြေစာFileကိုဖောက်သည်များ(customer)များအလွယ်တကူပေးပို့နိုင်ခြင်း
  • ပြင်ပ PDF readerမှလည်း PDF ပြေစာကိုဖတ်နိုင်ခြင်း
  • ထုတ်ကုန်ကို QR ကုဒ် သုံးပြီးထည့်ခြင်း
  • ဘားကုဒ်နံပါတ်ကို သုံး၍ ကုန်ပစ္စည်းကိုထည့်ခြင်း
  • ကုန်ပစ္စည်းသိုလှောင်မှုစီမံခန့်ခွဲမှုစနစ်ပါ၀◌င်ခြင်း
  • ထုတ်ကုန်ကို QR နှင့် Barcode သုံး၍ ရှာဖွေခြင်း
  • Cart စနစ်အသုံးပြုခြင်း
  • အသုံးစရိတ်များထည့်သွင်း၊ပြင်ဆင်နိုင်ခြင်း။
  • ထုတ်ကုန်အမျိုးအစား ပေါင်းထည့် / တည်းဖြတ် နိုင်ခြင်း
  • ငွေပေးချေမှုနည်းလမ်းများကိုထည့်သွင်း / တည်းဖြတ် နိုင်ခြင်း
  • နေ့စဉ်၊ လစဉ်နှင့်နှစ်စဉ်မှတ်တမ်းများကို ဘားဇယားဖြင့်ဖော်ပြခြင်း
  • ဒေတာဘေ့စ်အားBackup နှင့် Restoreလုပ်နိုင်ခြင်း
  • မည်သည့်ငွေကြေးသင်္ကေတကိုမဆို အသုံးပြုနိုင်ခြင်း
  • ဆွဲဆောင်မှုရှိသောUI အသုံးပြုထားခြင်း
  • အခွန်နှင့်လျှော့စျေးစနစ်ပါ၀င်ခြင်း
  • Excel (xls) ဖိုင်မှထုတ်ကုန်အချက်အလက်များကိုထည့်သွင်းနိုင်ခြင်း
  • ထုတ်ကုန်အချက်အလက်များကို Excel (xls) ဖိုင်မှ ထည့်သွင်းနိုင်ခြင်း
  • Excel (xls) ဖိုင်မှဖောက်သည်များ(Customer)နှင့်ပေးသွင်းသူများ(Supplier)များ၏အချက်အလက်များကို ထည့်သွင်းနိုင်ခြင်း
  • ဖောက်သည်များ(Customer)နှင့်ပေးသွင်းသူများ(Supplier)များ၏အချက်အလက်များကို Excel ဖိုင်အဖြစ်ထုတ်ယူနိုင်ခြင်း
  • ရောင်းအားဒေတာအားလုံးကို Excel ဖိုင်ဖိုင်အဖြစ်ထုတ်ယူနိုင်ခြင်း
  • ကုန်ကျစရိတ်အားလုံးကိုဒေတာများကို Excel ဖိုင်အဖြစ်ထုတ်ယူနိုင်ခြင်း
  • ဒေတာဘေ့စ်အပြည့်အစုံကို Excel အဖြစ်ထုတ်ယူနိုင်ခြင်း
  • ဘာသာစကားမျိုးစုံ (အင်္ဂလိပ်၊ မြန်မာ) ကိုထောက်ပံ့ခြင်း။
  • Android 10,11 နှင့်ကိုက်ညီခြင်း
